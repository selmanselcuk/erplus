# Apple Premium Stil Dönüşümü

## Renk Değişiklikleri
- Primary Blue: `#3498DB` → `#007AFF` (Apple Blue)
- Dark Text: `#2C3E50` → `#000000` (Pure Black)
- Gray Text: `#7F8C8D` → `#8E8E93` (Apple Gray)
- Light Gray: `#95A5A6` → `#C7C7CC` (Apple Light Gray)
- Background: `#F5F5F5` → `#F8F9FA` (Softer White)
- Green: `#27AE60` → `#34C759` (Apple Green)
- Orange: → `#FF9500` (Apple Orange)
- Red: → `#FF3B30` (Apple Red)

## Icon Değişiklikleri
Material Icons → Cupertino Icons:
- `Icons.info_outline` → `CupertinoIcons.info_circle`
- `Icons.contact_phone` → `CupertinoIcons.phone`
- `Icons.account_balance` → `CupertinoIcons.doc_text`
- `Icons.storefront` → `CupertinoIcons.cart`
- `Icons.payments` → `CupertinoIcons.money_dollar`
- `Icons.note` → `CupertinoIcons.pencil`
- `Icons.business` → `CupertinoIcons.building_2_fill`
- `Icons.shopping_cart` → `Cupertino Icons.cart_fill`
- `Icons.arrow_back` → `CupertinoIcons.back`
- `Icons.save` → `CupertinoIcons.checkmark_alt`

## Stil Değişiklikleri
- Gradient header → Solid white background
- Border radius: 6-8px → 12-14px (more rounded)
- Box shadows: harsh → subtle (opacity 0.03-0.06)
- Font weight: bold → w600 (SF Pro style)
- Letter spacing: -0.08 to -0.5 (Apple typography)

## Button Stili
- Material IconButton → CupertinoButton
- Flat buttons → Rounded with 12px radius
- Primary button: Filled blue background
- Secondary: Transparent with blue text
